報告無法由標準的 JUnit 測試執行程序建置的 JUnit 測試類別。

JUnit 4 測試類別需要為 public 並包含一個 public 無實參建構函式或完全不包含建構函式（隱式的預設建構函式），且沒有其他 public 建構函式。 JUnit 3 測試類別需要為 public，並且需要 public 無實參建構函式或者帶有單個 String 類型參數的 public 建構函式（呼叫符合的超建構函式）。 否則，測試類別無法由標準的 JUnit 測試執行程序執行。

範例：


public class MyTest {

  private MyTest() {} // no-arg 建構函式為 private

  @Test
  public void testSomething() {
    assertEquals(1, 1);
  }
}
