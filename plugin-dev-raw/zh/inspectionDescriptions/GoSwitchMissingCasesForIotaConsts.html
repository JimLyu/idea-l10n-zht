當 iota 關鍵字出現在 const 宣告中時，報告 switch 語句中未處理的值。 要處理這些值，請考慮使用 case 或 default 子句。

Go 中的 iota 關鍵字會在 const 塊中建立一個整數序列。 雖然 Go 編譯器不需要處理所有 iota 值，但缺少其中一些值可能表示存在錯誤。

有關 iota 的更多資訊，請參閱 go.dev 上的 Iota。

範例：

const (
  a = iota
  b
  c = -5
)

func main() {
  v := 5
  switch v {
  case a:
    break;
  case b:
    break;
  }
}

IDE 會觸發檢查，因為 c 未由 switch 語句處理。 請注意，c 位於相同的 const 塊中，即使它不使用 iota。

快速修復選項是 Create 'case' clause for values（為值建立 'case' 子句）和 Create 'default' clause（建立 'default' 子句）。 套用 Create 'default' clause 快速修復後：

const (
  a = iota
  b
  c
)

func main() {
  v := 5
  switch v {
  case a:
    break;
  default:
  }
}
