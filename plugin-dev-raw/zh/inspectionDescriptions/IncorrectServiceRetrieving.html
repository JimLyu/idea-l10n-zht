報告檢索服務時出現的以下問題：
  • 嘗試檢索未註冊的服務。
  • 檢索服務時不符合：嘗試將專案級服務作為應用程式級服務獲取，或者反向動作。

範例 (Kotlin)：


@Service
class MyAppService

@Service(Service.Level.PROJECT)
class MyProjectService(private val project: Project)


// 錯誤：
val projectService = service<MyProjectService>()         // 專案級服務作為應用程式級服務進行檢索
val applicationService = project.service<MyAppService>() // 應用程式級服務作為專案級服務進行檢索


// 正確：
val projectService = project.service<MyProjectService>()
val applicationService = service<MyAppService>();
