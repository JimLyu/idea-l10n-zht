報告可以取代為一般字串字面量的文本塊。 此檢查有助於為向後相容早期 Java 版本而降級。

範例：


  Object obj = engine.eval("""
                           function hello() {
                             print('"Hello, world"');
                           }

                           hello();
                           """);

在套用快速修復後：


  Object obj = engine.eval("function hello() {\n" +
                           "  print('\"Hello, world\"');\n" +
                           "}\n" +
                           "\n" +
                           "hello();\n");
