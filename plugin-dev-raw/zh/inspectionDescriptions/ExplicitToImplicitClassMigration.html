Reports ordinary classes, which can be converted into implicitly declared classes

範例：


  public class Sample {
      public static void main(String[] args) {
          System.out.println("Hello, world!");
      }
  }

在套用快速修復後：


  public static void main(String[] args) {
      System.out.println("Hello, world!");
  }
