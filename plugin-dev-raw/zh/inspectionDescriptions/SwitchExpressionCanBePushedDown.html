Reports switch expressions and statements where every branch has a common subexpression, and the switch can be moved inside. 此動作會縮短程式碼。 在許多情況下，將得到的 switch 表達式提取到單獨的變數或方法是合理的。

範例：


  switch (value) {
    case 0 -> System.out.println("zero");
    case 1 -> System.out.println("one");
    case 2, 3, 4 -> System.out.println("few");
    default -> System.out.println("many");
  }

在套用快速修復後：


  System.out.println(switch (value) {
    case 0 -> "zero";
    case 1 -> "one";
    case 2, 3, 4 -> "few";
    default -> "many";
  });

This inspection is applicable only for enhanced switches with arrow syntax.
