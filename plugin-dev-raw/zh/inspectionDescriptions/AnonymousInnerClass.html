報告匿名類別。

偶爾將匿名類別取代為內部類別可以提高程式碼的可讀性和可維護性。 某些程式碼標準不建議使用匿名類別。

範例：


  class Example {
    public static void main(String[] args) {
      new Thread() {
        public void run() {
          work()
        }

        private void work() {}
      }.start();
    }
  }

在套用快速修復後：


  class Example {
    public static void main(String[] args) {
      new MyThread().start();
    }

    private static class MyThread extends Thread {
      public void run() {
        work();
      }

      private void work() {}
    }
  }
