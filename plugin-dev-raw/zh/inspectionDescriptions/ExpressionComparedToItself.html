報告左動作數和右動作數表示相同表達式的比較。 雖然有時可能會有意將表達式與其自身進行比較，但通常是疏忽大意所致。

範例：


  // 可能 left.getLength() == right.getLength() 是有意為之
  boolean result = left.getLength() == left.getLength();

要忽略可能產生副作用的比較，請使用忽略有副作用的條件 選項。 停用此選項可能會導致誤報，例如，當同一方法在後續呼叫中返回不同的值時。

範例：


  native int unknownMethod();
  
  ...
  
  if (unknownMethod() > unknownMethod()) {
    System.out.println("Got it");
  }

由於 unknownMethod() 可能存在副作用（在範例中），只有在停用 忽略有副作用的條件 選項時才會觸發警告。
