報告可以更有效地編寫為 do-while 迴圈的 while 迴圈。 There are while loops where the code just before the loop is identical to the code in the body of the loop. Replacing with a do-while loop removes the duplicated code. For while loops without such duplicated code, the quick fix is offered in the editor as well, but without highlighting.

範例：


  foo();
  while (x) {
      foo();
  }

可以取代為：


  do {
    foo();
  } while (x);
