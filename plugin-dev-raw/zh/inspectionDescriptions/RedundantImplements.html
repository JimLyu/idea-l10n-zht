Reports interfaces in a class' implements list or an interface's extends list that are already implemented by a superclass or extended by a superinterface of that class. 此類別宣告不必要，並且可以安全移除。

範例：


  class X implements One, Two {
  }
  interface One {}
  interface Two extends One {}

在套用快速修復後：


  class X implements Two {
  }
  interface One {}
  interface Two extends One {}

Use the options to not report on Serializable or Externalizable in an extends orimplements list.
