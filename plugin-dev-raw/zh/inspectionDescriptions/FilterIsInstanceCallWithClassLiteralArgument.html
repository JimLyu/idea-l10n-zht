報告對帶有類別文字實參的 Kotlin 標準庫函數 filterIsInstance 的呼叫。 更習慣的做法是使用帶有具體化類型參數的此函式版本，以避免使用 ::class.java 語法。

注意：不報告對泛型類別文字的檢查，因為類型實參列表中的 Class<*, *> 語法可能不受歡迎。

範例：


  fun foo(list: List<*>) {
      list.filterIsInstance(Int::class.java)
  }

在套用快速修復後：


  fun foo(list: List<*>) {
      list.filterIsInstance<Int>()
  }
