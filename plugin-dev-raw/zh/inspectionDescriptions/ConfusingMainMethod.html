報告命名為 "main" 但沒有 public static void main(String[]) 簽名的方法。 Additionally main methods located in anonymous or local classes are reported. Anonymous and local classes do not have a fully qualified name and thus can't be run.

此類別方法可能令人困惑，因為命名為 "main" 的方法應為應用程式入口點。

範例：


  class Main {
    void main(String[] args) {} // warning here because there are no "public static" modifiers
  }

只能在編輯器中使用重新命名此類別方法的快速修復。
