報告 System.out 或 System.err 的用法。

此類別語句通常用於臨時偵錯，應當從生產程式碼中移除，或者取代為更穩健的日誌記錄工具。

提供的快速修復支援 SLF4J 和 Log4j 2。 它會將 System.out 和 System.err 呼叫取代為日誌呼叫

範例：


  public static void test(Object o) {
    System.out.println("Test: "+ o);
  }

在套用快速修復後：


  private static final Logger log = LoggerFactory.getLogger(Test.class);

  public static void test(Object o) {
      log.info("Test: {}", o);;
  }

使用 'System.err' 的日誌方法選項可以指定用於為 'System.err' 呼叫記錄訊息的方法。 使用 'System.out' 的日誌方法選項可以指定用於為 'System.out' 呼叫記錄訊息的方法。
