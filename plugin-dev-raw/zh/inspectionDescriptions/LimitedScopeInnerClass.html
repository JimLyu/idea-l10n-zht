報告本地類別。

本地類別是指在程式碼塊中宣告的命名嵌套類別。 本地類別不常見，因此可能令人困惑。 此外，有些程式碼標準不建議使用本地類別。

範例：


  class Example {
    void test() {
      class Local { // 這裡
      }
      new Local();
    }
  }

在套用快速修復後：


  class Example {
    void test() {
      new Local();
    }

    private static class Local { // 這裡
    }
  }
