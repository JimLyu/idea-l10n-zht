報告可以設定為 static 的內部類別。

static 內部類別不保留對其封閉實例的隱式參照。 When using Java 17 or before, this prevents a common cause of memory leaks and uses less memory per instance of the class.

範例：


  public class Outer {
    class Inner { // 非 static
      public void foo() {
        bar("x");
      }

      private void bar(String string) {}
    }
  }

在套用快速修復後：


  public class Outer {
    static class Inner {
      public void foo() {
        bar("x");
      }

      private void bar(String string) {}
    }
  }
