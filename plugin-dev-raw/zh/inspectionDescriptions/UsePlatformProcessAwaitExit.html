報告協程上下文中 Process.waitFor() 和 Process.onExit() 的用法。 禁止在協程上下文中使用這些方法。

請改用 com.intellij.util.io.awaitExit()。

範例：


suspend fun doSomething(process: Process) {
  val exitCode = process.waitFor() // 錯誤
  // ...
}

suspend fun doSomething(process: Process) {
  val exitCode = process.awaitExit() // 正確
  // ...
}
