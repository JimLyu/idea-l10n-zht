報告 instanceof 表達式，其中選中的表達式具有既不擴展/實作 instanceof 表達式右側的類別/接口類型，也沒有進行上述動作的子類別。

儘管它可以用於庫程式碼，但此類別建構很可能是錯誤的，因為專案中宣告的任何類別的任何實例都無法通過此 instanceof 測試。

範例：


  class Foo { }

  interface Bar { }
  
  class Main {
      void test(Foo f, Bar b) {
          if (f instanceof Bar) { // 問題
              System.out.println("fail");
          }
          if (b instanceof Foo) { // 問題
              System.out.println("fail");
          }
      }
  }
