報告未被保護條件包圍的日誌記錄呼叫。 對呼叫的實參進行評估可能開銷巨大。 當日誌記錄語句使用的級別停用日誌記錄時，使用臨界子句包圍日誌記錄呼叫可避免這種開銷。 這對於最不嚴重的日誌記錄呼叫級別（追蹤、偵錯、最精細）特別有用，因為這些呼叫在生產環境中最常被停用。

範例：


  public class TestObject {
    void test(Object object) {
      LOG.debug("some logging " + expensiveCalculation(1));
    }
  }

在套用快速修復後：


  public class TestObject {
    void test(Object object) {
      if(LOG.isDebugEnabled()){
        LOG.debug("some logging " + expensiveCalculation(1));
      }
    }
  }

此檢查支援 Log4j2 和 SLF4J 日誌記錄框架（ 建置器除外）。
