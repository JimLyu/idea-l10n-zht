Reports logging calls that are not surrounded by a guard condition. The evaluation of a call's arguments can be expensive. 當日誌記錄語句使用的級別停用日誌記錄時，使用臨界子句包圍日誌記錄呼叫可避免這種開銷。 這對於最不嚴重的日誌記錄呼叫級別（追蹤、偵錯、最精細）特別有用，因為這些呼叫在生產環境中最常被停用。

範例：


  public class TestObject {
    void test(Object object) {
      LOG.debug("some logging " + expensiveCalculation(1));
    }
  }

After a quick-fix is applied:


  public class TestObject {
    void test(Object object) {
      if(LOG.isDebugEnabled()){
        LOG.debug("some logging " + expensiveCalculation(1));
      }
    }
  }

This inspection supports Log4j2 and the SLF4J logging frameworks (except builders).
