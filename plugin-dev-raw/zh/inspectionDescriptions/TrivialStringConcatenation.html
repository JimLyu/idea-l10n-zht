報告字串串聯中的空字串動作數。 與空字串的串聯可以用於將非 String 物件或基元轉換為 String，但使用 String.valueOf() 方法呼叫會更清晰。

建議通過快速修復來簡化串聯。

範例：


  void foo(int x, int y) {
    String s = "" + x + " ; " + y;
  }

在套用快速修復後：


  void foo(int x, int y) {
    String s = x + " ; " + y;
  }

使用僅報告無需其他更改即可移除空字串的情況選項可以忽略移除空字串需要新增另一個動作數的 String.valueOf() 轉換的情況。
