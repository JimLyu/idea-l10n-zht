當用 @override 裝飾的方法在其上級類別中沒有符合的方法時報告

範例：


from typing import override

class Parent:
    def foo(self) -> int:
        return 1

    def bar(self, x: str) -> str:
        return x

class Child(Parent):
    @override
    def foo(self) -> int:
        return 2

    @override # 覆寫函式缺少 super 方法
    def baz(self) -> int:
        return 1
