報告未根據當前程式碼樣式排列的 import 語句（請參閱“設定|編輯|程式碼樣式”）。

範例：


  import java.util.List;
  import java.util.ArrayList;

  public class Example {
    List list = new ArrayList();
  }

在套用“最佳化 import”快速修復後： 


  import java.util.ArrayList;
  import java.util.List;

  public class Example {
    List list = new ArrayList();
  }
