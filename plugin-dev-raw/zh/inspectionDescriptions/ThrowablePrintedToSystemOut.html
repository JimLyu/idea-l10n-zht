報告以異常作為實參的 System.out.println() 呼叫。

使用 print 語句記錄異常會隱藏堆疊追蹤，這可能會使問題的調查變得複雜。 建議您改用記錄器。

也會報告以異常作為實參的 System.out.print()、System.err.println() 和 System.err.print() 呼叫。 建議使用記錄器來記錄異常。

The provided quick-fix supports SLF4J and Log4j 2. It replaces System.out.println() call with log calls

範例：


  try {
      foo();
  } catch (Exception e) {
      System.out.println(e);
  }

在套用快速修復後：


  try {
      foo();
  } catch (Exception e) {
    log.error("e: ", e);
  }

Use the Log method for fix option to specify a method which it is used to log a message.
