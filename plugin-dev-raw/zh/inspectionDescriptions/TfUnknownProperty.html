報告塊中的未知屬性

問題範例：


resource "awscc_appflow_flow" "aws_flow1" {
  flow_name          = ""
  source_flow_config = ""
  trigger_config     = ""
  destination_flow_config_list {}
  tasks {}
  test = ""
}

在此範例中，塊中的 test 屬性未知。

更正範例：


resource "awscc_appflow_flow" "aws_flow1" {
  flow_name          = ""
  source_flow_config = ""
  trigger_config     = ""
  destination_flow_config_list {}
  tasks {}
}
