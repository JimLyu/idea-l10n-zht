檢測無法根據已知提供程序進行解析的資源/資料來源。 提供程序應在 required_providers 部分中指定，或者應來自 hashicorp 命名空間。

例如，下面的程式碼會導致錯誤： 


terraform {
  required_version = "1.1.3"
}
resource "abbey_demo" "demo" {
  email      = "user@mail.company"
  permission = "user"
}

Terraform 將嘗試為 abbey_demo 資源匯入 hashicorp/abbey 提供程序。

要修正此問題，我們需要指定準確的提供程序坐標： 


terraform {
  required_version = "1.1.3"
  required_providers {
    abbey = {
      source  = "abbeylabs/abbey"
      version = "0.2.9"
    }
  }
}
resource "abbey_demo" "demo" {
  email      = "user@mail.company"
  permission = "user"
}

有關詳情，請參閱 Terraform 文檔。
