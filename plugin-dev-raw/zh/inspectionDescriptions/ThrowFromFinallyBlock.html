報告 finally 塊內的 throw 語句。 雖然偶爾是有意為之，但此類別 throw 語句可能隱藏從 try 和 catch 塊拋出的異常，並使偵錯過程變得複雜。

範例：


  class Fascinating {
    void doTask(Task t) {
      try {
        t.activity();
      } finally {
        if (!t.finish()) {
          throw new IllegalStateException();
        }
      }
    }
  }

使用在可能拋出已宣告異常的每一處發出警告選項，可以在呼叫宣告了異常的方法時也發出警告。
