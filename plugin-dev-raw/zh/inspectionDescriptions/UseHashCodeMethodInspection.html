通知您什麼時候可以用 Long.hashCode 或 Double.hashCode() 方法取代按位動作。 它會檢測模式 (int)(x ^ (x >>> 32)) 的實例，其中 x 是類型為 long 的變數，或者是前一個 Double.doubleToLongBits() 呼叫的結果。 此取代可以縮短程式碼，提高程式碼的可讀性。

範例： 


int result = (int)(var ^ (var >>> 32));
套用快速修復後： 

int result = Long.hashCode(var);

僅當專案或模組的語言級別為 8 或更高時，此檢查才會報告。
