報告類似於 string(x) 的表達式的轉換，其中 x 是整數，但不是位元組或 rune。

不建議此類別轉換，因為它們返回 Unicode 程式碼點 x 的 UTF-8 表示，而不是人們可能預期的返回 x 的十進制字串表示。 此外，如果 x 表示無效的程式碼點，轉換將不能被靜態拒絕。

對於打算使用程式碼點的轉換，請考慮將其取代為 string(rune(x))。 否則， strconv.Itoa 及其等效函式將根據需要返回值的字串表示。

範例：

func main() {
 a := 1
 _ = string(a)
}

在套用將整數轉換為 rune 快速修復後：

func main() {
 a := 1
 _ = string(rune(a))
}
