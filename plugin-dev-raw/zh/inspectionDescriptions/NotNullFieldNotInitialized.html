報告建構函式中未初始化的註解為非 null 的欄位。

範例：

public class MyClass {
  private @NotNull String value;

  public void setValue(@NotNull String value) {
    this.value = value;
  }

  public @NotNull String getValue() {
    return value;
  }
}

此類別欄位可能違反非 null 約束。 在上例中，setValue 參數註解為非 null，但是如果沒有呼叫 setter，getValue 可能返回 null。
