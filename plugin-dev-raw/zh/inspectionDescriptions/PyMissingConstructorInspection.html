報告類別中缺少對 super 構造函式進行呼叫的情況。

範例：

class Fruit:
    def __init__(self):
        pass


class Pear(Fruit):
    def __init__(self):
        pass

Pear 類別在 __init__ 方法中應有 super 呼叫。

套用快速修復後，程式碼變為：

class Fruit:
    def __init__(self):
        pass


class Pear(Fruit):
    def __init__(self):
        super().__init__()
