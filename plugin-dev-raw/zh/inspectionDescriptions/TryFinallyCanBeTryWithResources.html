報告可能使用 Java 7 Automatic Resource Management（這樣不容易出錯）的 try-finally 語句。

可以通過快速修復將 try-finally 語句轉換為 try-with-resources 語句。

範例：


  PrintStream printStream = new PrintStream(fileName);
  try {
    printStream.print(true);
  } finally {
    printStream.close();
  }

通過快速修復將原因傳遞給建構函式：


  try (PrintStream printStream = new PrintStream(fileName)) {
    printStream.print(true);
  }
