報告可通過取代為文本塊來簡化的 String 串聯。

要求：

  • \n 發生兩次或多次。
  • 文本塊未串聯。

使用報告單個字串字面量選項可以醒目提示包含換行符的單個字面量。 即使停用此選項，該快速修復仍然可用。

範例：


  String html = "<html>\n" +
                "    <body>\n" +
                "        <p>Hello, world</p>\n" +
                "    </body>\n" +
                "</html>\n";

在套用快速修復後：


  String html = """
                <html>
                    <body>
                        <p>Hello, world</p>
                    </body>
                </html>
                """;
