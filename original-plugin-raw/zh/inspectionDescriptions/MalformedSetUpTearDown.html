报告不声明为 public、不返回 void 或接受实参的 JUnit 3 setUp() 或 tearDown() 方法。 此类 setUp() 或 tearDown() 方法很可能是无意中创建的，JUnit 测试运行程序不会执行这些方法。

示例：


  public class WeightyTest extends TestCase {
    private File file;
    private int setUp(String prefix) throws IOException {
      file = File.createTempFile(prefix, ".tmp");
      return 0;
    }

    // ... 测试转到此处
  }
