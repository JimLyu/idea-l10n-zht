报告协程上下文中 Process.waitFor() 和 Process.onExit() 的用法。 禁止在协程上下文中使用这些方法。

请改用 com.intellij.util.io.awaitExit()。

示例：


suspend fun doSomething(process: Process) {
  val exitCode = process.waitFor() // 错误
  // ...
}

suspend fun doSomething(process: Process) {
  val exitCode = process.awaitExit() // 正确
  // ...
}
