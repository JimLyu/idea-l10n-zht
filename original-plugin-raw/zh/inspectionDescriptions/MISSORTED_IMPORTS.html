报告未根据当前代码样式排列的 import 语句（请参阅“设置|编辑|代码样式”）。

示例：


  import java.util.List;
  import java.util.ArrayList;

  public class Example {
    List list = new ArrayList();
  }

在应用“优化 import”快速修复后： 


  import java.util.ArrayList;
  import java.util.List;

  public class Example {
    List list = new ArrayList();
  }
