报告以异常作为实参的 System.out.println() 调用。

使用 print 语句记录异常会隐藏堆栈跟踪，这可能会使问题的调查变得复杂。 建议您改用记录器。

也会报告以异常作为实参的 System.out.print()、System.err.println() 和 System.err.print() 调用。 建议使用记录器来记录异常。

提供的快速修复支持 SLF4J 和 Log4j 2。 它会将 System.out.println() 调用替换为日志调用

示例：


  try {
      foo();
  } catch (Exception e) {
      System.out.println(e);
  }

在应用快速修复后：


  try {
      foo();
  } catch (Exception e) {
    log.error("e: ", e);
  }

使用修正的日志方法选项可以指定用于记录消息的方法。
