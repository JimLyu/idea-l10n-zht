报告以异常作为实参的 System.out.println() 调用。

使用 print 语句记录异常会隐藏堆栈跟踪，这可能会使问题的调查变得复杂。 建议您改用记录器。

也会报告以异常作为实参的 System.out.print()、System.err.println() 和 System.err.print() 调用。 建议使用记录器来记录异常。

The provided quick-fix supports SLF4J and Log4j 2. It replaces System.out.println() call with log calls

示例：


  try {
      foo();
  } catch (Exception e) {
      System.out.println(e);
  }

在应用快速修复后：


  try {
      foo();
  } catch (Exception e) {
    log.error("e: ", e);
  }

Use the Log method for fix option to specify a method which it is used to log a message.
