报告非 public、非 static、返回值类型为 void 或在没有声明形参解析器的情况下接受实参的 JUnit 4 @BeforeClass 与 @AfterClass 注解方法和 JUnit 5 @BeforeAll 与 @AfterAll 注解方法。 此类方法很容易无意间创建，但是 JUnit 测试运行程序不会执行这些方法。

示例：


  public class MainTest {
    @BeforeClass
    String beforeClass(int i) { ... }
  }

在应用快速修复后：


  class MainTest {
    @BeforeClass
    public static void beforeClass() { ... }
  }
