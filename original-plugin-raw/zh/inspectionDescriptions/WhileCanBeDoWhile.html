报告可以更有效地编写为 do-while 循环的 while 循环。 There are while loops where the code just before the loop is identical to the code in the body of the loop. Replacing with a do-while loop removes the duplicated code. For while loops without such duplicated code, the quick fix is offered in the editor as well, but without highlighting.

示例：


  foo();
  while (x) {
      foo();
  }

可以替换为：


  do {
    foo();
  } while (x);
