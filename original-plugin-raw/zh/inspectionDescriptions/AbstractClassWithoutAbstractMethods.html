报告没有任何 abstract 方法的 abstract 类。 In most cases it does not make sense to have an abstract class without any abstract methods, and the abstract modifier can be removed from the class. If the class was declared abstract to prevent instantiation, it is often a better option to use a private constructor to prevent instantiation instead.

示例：


  abstract class Example {
    public String getName() {
      return "IntelliJ IDEA";
    }
  }

Use the option to ignore utility classes.
