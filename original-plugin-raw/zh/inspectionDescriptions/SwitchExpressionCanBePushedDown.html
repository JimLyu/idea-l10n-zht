Reports switch expressions and statements where every branch has a common subexpression, and the switch can be moved inside. 此操作会缩短代码。 在许多情况下，将得到的 switch 表达式提取到单独的变量或方法是合理的。

示例：


  switch (value) {
    case 0 -> System.out.println("zero");
    case 1 -> System.out.println("one");
    case 2, 3, 4 -> System.out.println("few");
    default -> System.out.println("many");
  }

在应用快速修复后：


  System.out.println(switch (value) {
    case 0 -> "zero";
    case 1 -> "one";
    case 2, 3, 4 -> "few";
    default -> "many";
  });

This inspection is applicable only for enhanced switches with arrow syntax.
