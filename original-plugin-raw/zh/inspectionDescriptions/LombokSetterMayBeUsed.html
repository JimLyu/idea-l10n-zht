报告可由 lombok @Setter 注解替换的标准 setter 方法。

示例：


  import java.util.Date;

  public class MyClass {
    /**
     * 处理日期。
     */
    private Date processDate;

    /**
     * 设置日期。
     *
     * @param The date
     */
    public void setProcessDate(Date param) {
      processDate = param;
    }
  }

在应用快速修复/清理后：


  import lombok.Setter;
  import java.util.Date;

  @Setter
  public class MyClass {
    /**
     * 处理日期。
     * -- SETTER --
     * 设置日期。
     *
     * @param The date
     */
    private Date processDate;
  }

它仅在配置 lombok 库时报告。 要移除 lombok，请参阅 lombok 网站。
