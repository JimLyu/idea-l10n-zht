报告检索服务时出现的以下问题：
  • 尝试检索未注册的服务。
  • 检索服务时不匹配：尝试将项目级服务作为应用程序级服务获取，或者反向操作。

示例 (Kotlin)：


@Service
class MyAppService

@Service(Service.Level.PROJECT)
class MyProjectService(private val project: Project)


// 错误：
val projectService = service<MyProjectService>()         // 项目级服务作为应用程序级服务进行检索
val applicationService = project.service<MyAppService>() // 应用程序级服务作为项目级服务进行检索


// 正确：
val projectService = project.service<MyProjectService>()
val applicationService = service<MyAppService>();
