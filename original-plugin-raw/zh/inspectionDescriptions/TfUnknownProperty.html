Reports unknown properties in blocks

问题示例：


resource "awscc_appflow_flow" "aws_flow1" {
  flow_name          = ""
  source_flow_config = ""
  trigger_config     = ""
  destination_flow_config_list {}
  tasks {}
  test = ""
}

在此示例中，块中的 'test' 属性未知。

更正示例：


resource "awscc_appflow_flow" "aws_flow1" {
  flow_name          = ""
  source_flow_config = ""
  trigger_config     = ""
  destination_flow_config_list {}
  tasks {}
}
