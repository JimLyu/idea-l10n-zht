报告可转换为 data object 的 object

如果 object 使用 java.io.Serializable 进行注解，则 data object 会自动生成 toString、equals、hashCode 和 readResolve

主要在两种情况下，您应该考虑将 object 转换为 data object。 第一种情况是当自定义 toString 返回类的名称时。 第二种情况是当 object 实现 java.io.Serializable 时

示例：


    object Foo {
        override fun toString(): String = "Foo"
    }

在应用快速修复后：


    数据对象 Foo

仅当项目或模块的 Kotlin 语言级别为 1.8 或更高时，此检查才会报告
