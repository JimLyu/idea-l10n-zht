报告命名为 "main" 但没有 public static void main(String[]) 签名的方法。 Additionally main methods located in anonymous or local classes are reported. Anonymous and local classes do not have a fully qualified name and thus can't be run.

此类方法可能令人困惑，因为命名为 "main" 的方法应为应用程序入口点。

示例：


  class Main {
    void main(String[] args) {} // warning here because there are no "public static" modifiers
  }

只能在编辑器中使用重命名此类方法的快速修复。
