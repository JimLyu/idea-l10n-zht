Reports interfaces in a class' implements list or an interface's extends list that are already implemented by a superclass or extended by a superinterface of that class. 此类声明不必要，并且可以安全移除。

示例：


  class X implements One, Two {
  }
  interface One {}
  interface Two extends One {}

在应用快速修复后：


  class X implements Two {
  }
  interface One {}
  interface Two extends One {}

Use the options to not report on Serializable or Externalizable in an extends orimplements list.
