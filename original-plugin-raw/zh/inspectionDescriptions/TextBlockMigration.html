报告可通过替换为文本块来简化的 String 串联。

要求：

  • \n 发生两次或多次。
  • 文本块未串联。

Use the Report single string literals option to highlight single literals containing line breaks. The quick-fix will still be available even when this option is disabled.

示例：


  String html = "<html>\n" +
                "    <body>\n" +
                "        <p>Hello, world</p>\n" +
                "    </body>\n" +
                "</html>\n";

在应用快速修复后：


  String html = """
                <html>
                    <body>
                        <p>Hello, world</p>
                    </body>
                </html>
                """;
