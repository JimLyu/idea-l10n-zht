报告可通过替换为文本块来简化的 String 串联。

要求：

  • \n 发生两次或多次。
  • 文本块未串联。

使用报告单个字符串字面量选项可以高亮显示包含换行符的单个字面量。 即使禁用此选项，该快速修复仍然可用。

示例：


  String html = "<html>\n" +
                "    <body>\n" +
                "        <p>Hello, world</p>\n" +
                "    </body>\n" +
                "</html>\n";

在应用快速修复后：


  String html = """
                <html>
                    <body>
                        <p>Hello, world</p>
                    </body>
                </html>
                """;
