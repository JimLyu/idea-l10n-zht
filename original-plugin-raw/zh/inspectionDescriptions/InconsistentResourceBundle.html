报告资源包中包含的属性文件中的问题。
  • 报告失踪的翻译

    使用此选项可以报告在父属性文件中包含但在继承中缺失的的属性（除非它是语言方言）。

    示例：

    
    # messages.properties
    abc=xxx

    # messages_fr.properties
    # 空文件
    空属性 abc 将报告为未翻译。


  • 报告不一致的属性

    使用此选项可以报告在继承属性文件中包含但在父级（如果没有父级，则在同级）中缺失的属性。

    示例：

    
    # messages.properties
    # 空文件

    # messages_fr.properties
    abc=xxx
    此处的属性 abc 翻译不适用于法语以外的任何语言，因此，将在（默认）属性文件 messages.properties 中报告为缺失。


  • 报告使用相同值重写的属性

    使用此选项可以报告逐字复制粘贴到多个属性文件中的属性。


    示例：

    
    # messages.properties
    abc=xxx

    # messages_fr.properties
    abc=xxx
    属性 abc 将在文件 messages_fr.properties 中报告为不必要地继承。


  • 报告被不同的占位符重写的属性

    使用此选项可检查重写属性中的占位符一致性。

    示例：

    
    # messages.properties
    qwe={0}xxx{1}
    abc={0}yyy{1}

    # messages_fr.properties
    qwe={0}xxx{0}xxx{1}
    abc={0}yyy
    将报告属性 abc，因为属性包含值与 messages.properties 不对应的消息格式占位符。


  • 报告使用不同值结尾重写的属性

    使用此选项可检查重写属性中的结尾一致性。

    示例：

    
    # messages.properties
    abc=xxxzzz

    # messages_fr.properties
    abc=xxx;
    属性 abc 将被报告为以特殊符号（! / ? / . / : / ;）结尾，而 messages.properties 中的父值则不会。
